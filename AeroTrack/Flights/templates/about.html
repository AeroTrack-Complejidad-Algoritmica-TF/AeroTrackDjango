Diseño Aplicativo

El aplicativo recibe como input los datos de nodos (aeropuertos) y aristas, con un peso heurístico que representa el tráfico máximo registrado, y otro que refleja el tráfico actual. A partir de estos datos, se busca la ruta más conveniente (la de menor tráfico) entre el nodo inicial y el final, según el vuelo. Durante el vuelo, se ajusta el peso de las aristas del grafo para reflejar el tráfico actual, calculado en función de la cantidad de aviones en la misma ruta. Finalmente, el camino mínimo puede calcularse usando el tráfico registrado o dejando que el algoritmo lo haga automáticamente mediante servicios.

Propuesta

Para resolver el problema de gestión del tráfico aéreo, se realizará un programa con el lenguaje de Python, la cual será capaz de leer 1500 datos de un dataset con la información de vuelos de realizados en Perú. El programa representará los datos de los vuelos en nodos de grafos y utilizará el algoritmo de A* (A Star), el cual permite escoger el mejor camino dentro de grafos, utilizando las heurísticas. Además, se utilizarán de apoyo diversas librerías, que nos permitirán graficar los grafos con sus nodos y administrarlos de manera correcta.
